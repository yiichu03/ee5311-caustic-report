# Task 3 可视化与主结果解读

## 运行设置
- 分辨率 N: 72
- 训练轮数 epochs: 300
- 学习率 lr: 0.01
- TV 正则权重: 0.005
- GIF 取帧间隔: 每 20 个 epoch 一帧
- 重点展示 pattern: EE5311

## 文件说明
- `*_main_result.png`: 主结果图，汇总 target、rendered、height map 和 difference heatmap。
- `*_surface_3d.png`: 三维表面图，展示优化得到的玻璃高度结构。
- `*_difference_heatmap.png`: 差值热图，观察误差主要分布区域。
- `*_loss_curve.png`: 损失曲线，观察优化稳定性。
- `*_optimization.gif`: 优化过程 GIF，展示生成图案逐步成形的过程。
- `metrics_summary.json`: 各 pattern 的数值指标汇总。

## 核心结果总结
- 优化在早期 epoch 会快速降低损失，随后进入缓慢收敛阶段，说明参数更新先完成大尺度能量重分配，再逐步修正局部细节。
- 主结果图显示，焦散亮度能够集中到目标字符或笔画主体，说明高度图学到的不是随意起伏，而是能够引导光线偏折的有效表面结构。
- 三维表面整体较平滑，没有大量尖锐毛刺，这表明 TV 正则化有效抑制了不稳定的高频振荡，也使结果更接近可制造的连续表面。
- 差值热图中误差通常出现在细笔画、边缘和拐角，这说明当前简化模型更擅长重建低频轮廓，对局部高精度结构仍有限制。
- GIF 可以直观看到亮斑从扩散态逐渐收缩到目标区域，说明优化过程具有明确方向性，而不是偶然得到最终结果。

## 适合直接放进博客或报告的总结段落
在学习率设为 0.01 的条件下，焦散优化能够稳定地把目标图案的主轮廓重建出来。主结果图和 GIF 共同表明，优化首先建立大尺度亮度分布，再逐步强化目标笔画。三维表面图说明最终学到的是连续、可解释的表面起伏结构，而不是不规则噪声。差值热图进一步表明，误差主要来自细节边缘和高曲率区域，这与当前模型采用的有限网格、Gaussian 光斑近似和简化折射传播假设是一致的。整体而言，这组结果足以支持“可微优化能够从目标光强分布反推出具有结构性的透镜表面”这一核心结论。
